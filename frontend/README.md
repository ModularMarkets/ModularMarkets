# Market Maker Frontend

React + TypeScript frontend for the Market Maker application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The frontend will run on `http://localhost:3000` and proxy API requests to `http://localhost:8000`.

## Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Development

Make sure the backend API is running on port 8000 before starting the frontend.
